# PUDIM
pudim is a derivated of ruby 
like:
java has kotlin
c has c++
ruby has PUDIM

## Example
ruby
```sh
puts "Hello World"
```
pudim
```sh
putln("Hello World");
```

## Automatic Installation
```sh
sudo apt install git ghc && git clone https://github.com/mioojo/pudim && cd pudim && chmod +x ./setup && ./setup 
```

## Commands
```sh
pudim-new ##Creates a new main.pud
pudim-run ##Run a main.pud
```
